## Problem

Computer vision models are often hard to inspect unless you are comfortable with scripts and raw model output. YOLOPv2-Studio turns that workflow into a more approachable interface for object detection, lane detection, and drivable-area segmentation demos.

## Approach

The project uses Python with Flask, OpenCV, PyTorch, and supporting scientific packages. The studio provides upload flows, demo media, webcam processing, configurable thresholds, task toggles, and an FPS display for practical model interaction.

## Outcome

This project is valuable because it broadens the portfolio beyond frontend-only apps. It shows practical AI-adjacent tooling: wrapping a computer vision model with a usable interface, documenting setup, and making visual model output easier to explore.
