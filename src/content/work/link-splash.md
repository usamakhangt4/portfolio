## Problem

Link-in-bio tools look simple on the public page, but the real product challenge lives in the dashboard: authentication, ordering, editing, previewing, and publishing all need to feel quick.

## Approach

LinkSplash uses Next.js with Firebase services for authentication and persistence. The dashboard supports drag-and-drop ordering, QR code generation, and public profile routes so users can move from setup to sharing without a heavy workflow.

## Outcome

This project is useful portfolio proof because it connects frontend polish with product mechanics: auth, database-backed state, interactive editing, and public route generation.
