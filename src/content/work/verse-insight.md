## Problem

Religious reference apps need clear reading surfaces and predictable navigation. Verse Insight is intentionally simple: a table-first interface where users can open rows and read verses.

## Approach

The app uses Next.js routes and local JSON data for Quran content. The work is best presented as a clean reading interface rather than a complex backend or data-platform project.

## Outcome

The deployed reader turns local surah data into a searchable table with expandable verse rows. The result is a compact reading interface that works without depending on a remote content API.
